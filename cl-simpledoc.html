External Symbols

*output-root* [Variable]
Default directory path where document-package will produce its output file.

document-package (package &optional outpath) [Function]
Documents given package in its own standalone HTML file at outpath if given, otherwise it makes a pathname from the package name and merges it with *output-root*

print-package-docs (package stream &key (external t) (internal nil) (functions nil) (macros nil) (generic-functions nil) (classes nil) (variables nil)) [Function]
Do a mass conversion of documentation from a package into HTML. Does NOT add proper HTML headers and footers; this way you can document more than one package into a single HTML file stream.
